# 🗳️ Blockchain Based Voting System

A secure and transparent voting system built using Blockchain technology. This project ensures that votes are immutable, tamper-proof, and verifiable, providing a trustworthy digital voting experience.

---

## 🚀 Features

- 🔐 Secure authentication for voters
- 🧾 Transparent and tamper-proof voting records using Blockchain
- 📊 Real-time vote counting
- 👤 One-person-one-vote system
- 🧠 Decentralized data storage
- 📱 Simple and user-friendly interface
- ⚡ Fast and efficient voting process

---

## 🛠️ Tech Stack

- HTML – Structure of the application

## 🧠 How It Works

1. Voters register/login into the system  
2. Each vote is recorded as a block in the blockchain  
3. Blocks are linked using cryptographic hashes  
4. Once added, votes cannot be modified or deleted  
5. Results are calculated transparently from the chain
